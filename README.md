# **TetrisGame**

## What this is
### This is tetris game made with python pygame

## How to use
### Runing the code
1. Install pygame
2. Copy and past the code(main.py) on you favorite Python IDE(ex. IDLE, VScode, PYcharm)
3. Run the code

### Playing the game
- Right : Right arrow key
* * *
- Left : Left arrow key
* * *
- Down : Down arrow key
* * *
- Trun : Up arrow key
